
# あ행

이곳은 [[docs/language/japanaese/regular-use-kanji/ko_kr|일본의 상용한자]] 중 あ행에 해당되는 한자들에 대해 설명하는 곳이다. あ행에 해당되는 한자는 총 130개다. 그걸 다시 あいうえお, 총 5단으로 나누면 평균 26개가 된다. 그 정도면 충분히 커버할 만한 개수가 될 것이다.
