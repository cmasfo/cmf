
# cmf 리포지토리

`cmf`는 내가 주력으로 관리하는 리포지토리다.

하위에 여러 가지 프로젝트들을 가지고 있으며, 주요 소스 코드는 `src` 폴더에 들어가 있다.

실사용을 할 땐 `pub` 리포지토리도 `cmf` 리포지토리의 하위 폴더로 들어가게 되지만, 실질적으로는 별개의 리포이므로 `pub` 리포지토리에 관한 건 `pub` 페이지에서 설명한다.
